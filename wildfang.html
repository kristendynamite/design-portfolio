Wildfang Product Page

Single page showcasing a product for sale for the brand Wildfang, as part of an Epicodus independent project demonstrating an understanding of CSS basics. Includes a list of product features, a product image that contains an absolutely positioned element, a table for product sizing, and a gallery of smaller related images. Floats, pseudo-elements, selectors, and various CSS elements are used.
